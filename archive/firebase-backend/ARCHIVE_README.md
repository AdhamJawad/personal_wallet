# Firebase Backend Archive

This directory contains the archived Firebase backend implementation for Personal Wallet.

## Contents
- `functions/`: original Cloud Functions TypeScript source
- `firebase.json`: original Firebase deployment configuration
- `.firebaserc`: original Firebase project mapping
- `firestore.rules`: original Firestore security rules
- `storage.rules`: original Storage security rules

## Archive Intent
- Reference only
- Do not delete during Laravel migration
- Treat as the executable specification for current backend behavior
